Truck Tracker

Truck Tracker is a modern SwiftUI application designed for food truck operators to monitor orders, track sales performance, analyze popular menu items, and manage daily operations from a single dashboard.

Built using Apple’s latest frameworks, Truck Tracker delivers a responsive experience across iPhone, iPad, and Mac from a unified codebase.

Features

Order Management

* Track incoming and active orders
* Monitor order preparation status
* View detailed order information
* Manage completed orders

Sales Analytics

* Interactive charts for sales insights
* Track top-performing menu items
* Visualize revenue trends
* Analyze customer ordering patterns

Dashboard

* Real-time business overview
* Pending order monitoring
* Performance metrics
* Quick access to operational data

Weather Integration

* View destination weather forecasts
* Access current weather conditions
* Support for route and location planning

Modern SwiftUI Experience

* NavigationSplitView architecture
* Swift Charts visualizations
* Adaptive layouts for iPhone, iPad, and Mac
* Light and Dark Mode support

Widgets & Live Activities

* Home Screen widgets
* Lock Screen updates
* Live order tracking
* Dynamic Island support (supported devices)

Technologies

* Swift 6
* SwiftUI
* NavigationSplitView
* NavigationStack
* Swift Charts
* WidgetKit
* ActivityKit
* WeatherKit
* StoreKit
* Async/Await
* MVVM Architecture

Installation

Clone the repository:

git clone https://github.com/PoorneshMV/Food-Truck.git

Open the project:

cd Food-Truck
open "Food Truck.xcodeproj"

Or open the project manually in Xcode.

Requirements

* Xcode 16+
* iOS 18+
* macOS 15+
* Swift 6

Run

1. Open the project in Xcode.
2. Select the Food Truck scheme.
3. Choose an iPhone Simulator or physical device.
4. Build and Run (⌘ + R).
